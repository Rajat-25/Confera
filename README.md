# Confera

A real-time video and chat application built with Next.js, Turborepo, Prisma, Express, WebRTC, and WebSocket.

## Getting Started

## Prerequisites

- Node.js installed on your machine

## Installation

1. Clone the project

2. Install dependencies
```bash
cd confera
npm install
```

3. (IMPORTANT ) Add required environment variables to the .env file of each package/app. 

4. Generate prisma client and apply db migrations.



## Development

To run the development server:
```bash
npm run dev
```

## Build

To build for production:
```bash
npm run build
```
